# Менеджер закупок Luzi

Веб-приложение для управления совместными закупками масел Luzi.

## 🚀 Как запустить на Render

1. Загрузи все файлы в GitHub репозиторий
2. Зайди на [Render.com](https://render.com)
3. Нажми **New +** → **Web Service**
4. Подключи GitHub репозиторий
5. Настройки:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `uvicorn app:app --host 0.0.0.0 --port $PORT`
6. Нажми **Deploy**

Приложение будет доступно по ссылке вида `https://aroma-manager.onrender.com`

## 📋 Как пользоваться

### Первый запуск
1. Открой приложение
2. Перейди в раздел «Покупатели» — добавь всех покупателей с телефонами и адресами
3. Нажми «Новая закупка» — вставь ссылку на Google Sheets с заказами

### Работа с закупкой
- **Розлив:** подруга сортирует по аромату, отмечает что разлито
- **Отправка:** подруга видит всё что нужно упаковать по покупателям
- **Оплата:** сводная таблица со статусами оплат и отправки

### Заказы с наличия
- Нажми «+ Наличие» чтобы добавить дозаказ
- Можно привязать к активной закупке или оставить свободным

## 🔄 Обновление статусов
Все статусы сохраняются мгновенно при клике на чекбокс.
Страница автоматически обновляется через 0.5 секунды.

## 📊 Структура Google Sheets
Ожидаемые колонки:
- Имя покупателя
- Название аромата
- Объём (мл)
- Цена за 10мл
- Сумма
